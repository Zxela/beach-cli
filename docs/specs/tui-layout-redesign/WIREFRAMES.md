## Section Specifications

### Weather Section
- **Height:** 6 lines (header + 5 data lines)
- **Width:** Full container width
- **Content:** Condition icon, temp, feels-like, wind, humidity, UV, sunrise/sunset

### Tides Section (Collapsed)
- **Height:** 5 lines
- **Width:** Full container width
- **Content:** State + height, sparkline (full width), time markers, next high/low

### Tides Section (Expanded)
- **Height:** 12 lines
- **Width:** Full container width
- **Content:** ASCII graph with Y-axis (0-4m), X-axis (6AM-12AM), curve, current marker

### Hourly Forecast Section
- **Height:** 7-9 lines (header + 6-8 hours)
- **Width:** Full container width
- **Format per line:** `HH:00   XXÂ°C  icon  Wind: XXkm/h  UV: X`

### Water Quality Section
- **Height:** 3-4 lines
- **Width:** Full container width
- **Content:** Status icon + text, last tested date, E.coli count

### Best Window Section
- **Height:** 6-8 lines (when activity selected)
- **Width:** Full container width
- **Content:** Top 3 time windows with scores, reasons, factor bars
